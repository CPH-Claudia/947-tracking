# 947-tracking
Python data processing pipeline for tracking customer opportunity lists, visit-to-conversion performance, and Tableau dashboard inputs.

## 一、專案概述

### 1. 專案目的

「947 商機客戶名單追蹤報表」用於追蹤商機名單釋出後的：

- 名單規模與分配情況
- 業務拜訪及標籤登打情況
- 客戶成交件數、FYC 與件均績效
- 各包名單每週新增成果與累積成果
- 各類名單是否仍具有持續推動價值

> 
> **重要提醒：客戶名單是死資料，名單貼標後不會自動回收或更新**
> 
> 這五包名單是依照貼標當下的客戶狀態與業務歸屬產出。名單貼出後，即使客戶後續被其他業務接手、客戶狀態改變，或不再符合原本貼標條件，既有標籤也不會自動收回或轉移。
> 
> 因此，若某位客戶在貼標日後被其他業務接走，新接手業務不一定會在自己的名單中看到該客戶；報表呈現的是貼標當下的結果，而不是即時更新的客戶歸屬。
>
