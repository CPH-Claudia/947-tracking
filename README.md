# 947 商機客戶名單追蹤資料處理

本專案為 947 商機客戶名單追蹤報表的 Python 資料處理流程。
主要用於整理商機名單客戶的拜訪資料、成交資料與拜訪後成交關係，
最終輸出整理後資料供 Tableau 儀表板使用。

## 專案目的
追蹤五類商機客戶名單：
- 2026(上)儲蓄險最後繳費
- 2027(上)儲蓄險最後繳費
- 2026(下)儲蓄險最後繳費
- 2027(下)儲蓄險最後繳費
- 投資型潛在客

## 整體流程
1. 更新商機名單追蹤報表資料源
2. 更新標籤(直)與繳款 FYC 跳板資料源
3. 更新 拜訪_關鍵字.hyper
4. 執行 Python 拜訪後成交資料處理
5. 更新 Tableau 資料源
6. 於 Tableau 儀表板查看最終追蹤結果

## 主要輸入資料
- 商機名單資料
- 標籤(直)保單資料
- 拜訪資料
- 拜訪關鍵字資料
- 繳款 FYC 資料

## 主要輸出資料
- 拜訪後成交欄位資料
- Tableau 使用的中介 CSV
- Tableau Extract / Hyper 的前置資料

## 注意事項
本 repository 不包含任何真實客戶資料、保單資料、公司內部路徑與 Tableau 檔案。
